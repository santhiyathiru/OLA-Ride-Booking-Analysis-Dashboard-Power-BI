# 🚖 OLA Ride Booking Analysis Dashboard | Power BI

## 📌 Project Overview

The **OLA Ride Booking Analysis Dashboard** is an interactive Power BI project designed to analyze ride booking data and provide valuable business insights. The dashboard enables users to monitor booking performance, revenue, cancellations, customer ratings, and vehicle-wise analysis through interactive visualizations.

This project demonstrates the use of **Power BI, Power Query, DAX, and Data Modeling** to transform raw booking data into meaningful insights for better decision-making.

---

## 🛠️ Tools & Technologies

- Microsoft Power BI
- Power Query
- DAX (Data Analysis Expressions)
- Data Modeling
- Microsoft Excel (Dataset)

---

## 📊 Dashboard Pages

### 1️⃣ Overall Dashboard
Provides a high-level summary of business performance.

**KPIs**
- Total Bookings
- Successful Bookings
- Cancelled Bookings
- Cancellation Percentage
- Total Revenue
- Average Customer Rating

---

### 2️⃣ Vehicle Type Analysis

Analyzes booking performance across different vehicle categories.

Features:
- Vehicle-wise Bookings
- Revenue by Vehicle Type
- Booking Distribution
- Performance Comparison

---

### 3️⃣ Revenue Analysis

Provides revenue-related insights.

Features:
- Total Revenue
- Revenue Distribution
- Revenue by Vehicle Type
- Revenue Trends

---

### 4️⃣ Cancellation Analysis

Tracks booking cancellations.

Features:
- Total Cancelled Bookings
- Cancellation Percentage
- Driver Cancellation
- Customer Cancellation
- Cancellation Breakdown

---

### 5️⃣ Ratings Analysis

Evaluates customer satisfaction.

Features:
- Average Ratings
- Rating Distribution
- Service Performance
- Customer Feedback Insights
